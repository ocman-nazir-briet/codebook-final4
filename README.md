## Usage

Installing requirements and dependencies
```
git clone https://github.com/ocman-nazir-briet/codebook-final4.git
cd codebook-final4
pip install -r requirements.txt
```

Running the server
```
python manage.py
```
OPening the Server
```
127.0.0.1:8000
```
